# react-native-appay-mir

Fork of `react-native-appay` with MIR payments network support

## Installation

```sh
npm install react-native-appay-mir
```

## Usage

```js
import { multiply } from 'react-native-appay-mir';

// ...

const result = await multiply(3, 7);
```
